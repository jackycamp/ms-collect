ms_collect.utils.Utils.to_mspoints(rawpoints: List[Tuple[float]] = [], sort_by_intensity_asc: bool = False, sort_by_intensity_desc: bool = False) → List[ms_collect.point.Point]¶

Converts a list of tuples that contain floats into a list of Points.

Args:

rawpoints (List[Tuple[float]], optional): The list of raw data. Defaults to []. sort_by_intensity_asc (bool, optional): Whether or not to have the returned list of points sorted by increasing intensity. Defaults to False. sort_by_intensity_desc (bool, optional): Whether or not to have the returned list of points sorted by decreasing intensity. Defaults to False.

Returns:

List[Point]: the converted list of Points.
