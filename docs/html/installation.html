InstallationÂ¶

ms-collect is a standard pip package that can be found in the Python Package Index.

You can install this package using pip:

$ pip install ms-collect

ms-collect requires Python 3.7 and above. To ensure that you install this module with Python 3 you can use this alternative:

$ pip3 install ms-collect
